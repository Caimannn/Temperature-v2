# Strategy Notes

This bot starts with a narrow scope.

- Market type: highest temperature only.
- Horizons: today and tomorrow only.
- Output: text-only position advice.
- Advice types: HOLD, CLOSE, ADD, BUY NEW, SWITCH.
- SWITCH means close the current position and open a different one.
- No trading logic is enabled in the initial foundation.

The first version should only prepare data and signals for later review.
